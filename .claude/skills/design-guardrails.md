DESIGN GUARDRAILS

Source Of Truth

Desktop is the master design.

Mobile adapts desktop.

Mobile does not replace desktop.

⸻

Forbidden Actions

Never:

* redesign sections
* remove components
* replace layouts
* introduce new visual systems

unless explicitly requested.

⸻

Responsive Rules

When fixing mobile:

Do:

* reflow content
* adjust spacing
* adjust sizing
* adjust hierarchy

Do not:

* create new UX paradigms
* invent new navigation systems
* replace interactions

⸻

Visual Consistency

Preserve:

* typography
* spacing philosophy
* visual rhythm
* image treatment
* card language

⸻

Design Debt Prevention

Before changing layout:

Evaluate:

1. Can spacing solve this?
2. Can sizing solve this?
3. Can hierarchy solve this?

Only redesign if all fail.

⸻

Portrait Rules

Portraits are identity assets.

Do not:

* remove
* replace
* relocate drastically

without explicit approval.

⸻

Content Density

Do not solve density issues by deleting content.

Prefer:

* grouping
* stacking
* progressive disclosure

Content removal requires approval.
