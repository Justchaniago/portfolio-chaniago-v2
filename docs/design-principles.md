# Design Principles

## Visual Philosophy

- Premium over flashy
- Editorial over corporate
- Sophisticated over trendy
- Timeless over fashionable
- Minimal but never empty

## Motion Philosophy

- Motion communicates hierarchy
- Motion communicates state change
- Motion must never be decorative
- Every animation must have purpose
- Motion should feel physical and intentional

## Layout Philosophy

- Strong focal points
- Clear narrative flow
- Large breathing room
- Consistent rhythm
- No visual clutter

## Interaction Philosophy

- Immediate response
- Smooth transitions
- No abrupt changes
- No surprising behavior
- Predictable but delightful

## Portfolio Philosophy

- Showcase thinking, not only visuals
- Guide users through a story
- Projects are chapters
- Work is presented as an experience
- Avoid portfolio grid mentality

## Performance Philosophy

- 60fps target
- GPU accelerated transforms
- Minimize layout shifts
- Minimize repaint costs
- Respect mobile performance
